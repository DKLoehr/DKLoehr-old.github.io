Complex Paint

A program that allows visualizations of functions on Complex numbers by mapping the input and output planes side-by-side. It allows various shapes to be painted to visualize the images of entire regions as well. Finally, one may iterate a function arbitrarily many times, determining fixpoints or generating Julia sets.
Complex Paint is written in C++, using the SFML library. It is a recreation of a very old program of the same name by Marc Parmet.



FractaSketch

A program that allows the creation of various fractal forms by repeatedly iterating a template. A fractal template specifies how to replace a line with a sequence of (typically smaller) lines. The process is then repeated on each smaller line, and so on until infinity. The result is a fractal shape.
FractaSketch is written in C++, using the SFML library. It is a recreation of a very old program of the same name by Peter Van Roy.
